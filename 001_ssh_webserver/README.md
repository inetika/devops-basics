# SSH
## Create ssh key
```
ssh-keygen
```
## Get contents of the public key
```
cat ~/.ssh/id_rsa.pub
```
## SSH into the server
```
ssh ServerIP
```

# Navigation in Shell

* `pwd` - print working directory
* `ls` - list directory's contents
* `cd` - change directory
* `cat` - show contents of a file
